# Précisions sur les extractions

## Données invalides et d'absence
Chaque script précise dans la boite de dialogue les points d'attention, données exclues ou incluses. En effet, les données d'absence ou invalides sont intégrées dans l'extraction de données brutes mais exclues des synthèses (tableaux, cartes).

## Codes atlas
Les données étant récupérées par l'API de Biolovision, les codes atlas utilisées sont ceux mis à disposition par l'API.
La liste et les correspondances de ces codes, figurent ici : https://wiki.biolovision.net/index.php/Correspondance_codes_atlas. Les colonnes `Codes Biolovision`, `Codes EBCC` et `Texte FR` sont à prendre en référence.
